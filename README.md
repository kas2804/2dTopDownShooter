This project is a 2D top-down shooter developed in
Unity. The player navigates using WASD controls to aim and
shoot projectile prefabs at incoming enemies. The game features
a Follow Enemy AI script that chases the player, alongside a
modular health system that manages damage and destruction for
all characters.
